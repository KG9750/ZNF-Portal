# Codex Auto Dev Pipeline

## Rules

For every GitHub Issue:

1. Create branch:
   feature/task-{id}

2. Implement ONLY the scope of the issue

3. Commit format:
   feat(task-{id}): implement feature

4. Push branch

5. Open Pull Request

6. Link PR to Issue

---

## Forbidden
- No extra features
- No architecture changes
- No combining multiple issues
